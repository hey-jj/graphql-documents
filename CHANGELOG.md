# Changelog

## [0.2.0] - 2026-07-07

### Changed
- Inline fragments with type names that share a prefix now sort by the full type name before their selection contents. (#12)
- Empty selection sets now return a parse error. (#13)
- Empty variable and argument lists now return a parse error. (#14)
- Spread punctuators must now use three adjacent dots, so whitespace or comments between dots return a parse error. (#15)
